# 사이트 구성 계획(요약)

공통 레이아웃
- 상단 좌측: 홈으로 돌아가는 버튼(Pargame's Blog)
- 상단 우측: Graph 링크(필요 시 확장)
- 하단: 저작권 표기, 이메일(클로킹), 버전/커밋

홈(index.html)
- posts/의 포스트만 최신순 20개 노출
- README/Index 류 자동 제외

그래프(graph.html)
- Posts 제외, 아카이브/노트 전용
- 아카이브/토픽 필터(단일 토픽 선택), URL 동기화
- 문서 목록과 뷰어 iframe 제공

뷰어(viewer.html)
- 상단에 제목 · 날짜 · 작성자 메타 라인 추가
- front matter 제거 후 안전 렌더링(DOMPurify)
- [[링크]]를 동일 아카이브/토픽을 우선하여 라우팅
- 역링크 영역 제공

배포
- main push → GitHub Pages 자동 배포(.github/workflows/pages.yml)

앞으로의 개선안
- 검색(제목/본문/태그)
- 그래프 다중 토픽 교집합 필터
- 포스트 템플릿 스니펫/CLI
