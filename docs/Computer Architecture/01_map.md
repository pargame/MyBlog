# 구조 지도

이 문서는 컴퓨터 구조의 주요 부품과 계층, 데이터 흐름을 한눈에 파악할 수 있도록 구조 지도를 그리는 방법을 안내합니다. 실무에서 자주 쓰이는 다이어그램, 계층도, 데이터/제어 흐름도 작성 팁도 함께 제시합니다.

## 왜 구조 지도가 중요한가?
- 복잡한 시스템을 한눈에 파악, 병목·핫스팟·계층 구조를 빠르게 이해
- 설계/디버깅/성능 분석 시 전체 맥락을 놓치지 않게 도움

## 실무 팁
- 다이어그램은 너무 복잡하게 그리기보다, 계층별·흐름별로 나누어 그리면 효과적
- SVG/ASCII/손그림 등 다양한 방식 활용 가능
