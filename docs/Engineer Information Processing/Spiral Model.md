# Spiral Model



> 반복적 개발과 위험 관리를 결합한 대규모·고위험 프로젝트에 적합한 개발 모델. [[SDLC]]의 대표적 모델.

---
**특징 및 장단점**
- 계획 → 위험분석 → 개발 → 평가(반복)
- 위험 체계적 관리, 반복적 요구 반영
- 복잡성↑, 비용·시간 증가
