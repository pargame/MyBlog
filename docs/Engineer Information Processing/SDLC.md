# 소프트웨어 개발 생명주기(SDLC)

소프트웨어 개발 생명주기(SDLC)는 소프트웨어를 계획, 개발, 테스트, 배포, 유지보수하는 전 과정을 단계별로 정의한 체계적 절차이다.




> 소프트웨어 개발의 전 과정을 단계별로 체계화한 모델. 각 단계별 산출물과 절차를 명확히 하여 품질·일정·비용을 관리함. [[SE]]와 밀접한 관계.

---
**주요 단계**
- 요구사항 정의([[Requirement]])
- 요구사항 분석([[RequirementAnalysis]])
- 설계
- 구현
- 테스트
- 유지보수

**대표 모델**
- [[Waterfall Model]]
- [[Prototype Model]]
- [[Agile Model]]
- [[Spiral Model]]

**관련 문서**: [[SE]]
