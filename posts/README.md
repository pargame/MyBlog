# Posts

Write your Markdown posts here. Examples:

- posts/2025/hello-world.md
- posts/notes/some-topic.md

Notes
- The first folder segment (e.g., `2025`) becomes the archive.
- All folder names become topics. You can also add tags in frontmatter, for example:
	# ---
	# title: Hello
	# date: 2025-08-13
	# tags: [blog, intro]
	# ---

- Wiki links like [[SomeNote]] or [[SomeNote|nice title]] are supported.
- Homepage lists recent items within the Posts collection.
